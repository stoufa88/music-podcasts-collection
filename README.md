# Podcasts collection

Listening to music everyday on different channels such as soundcloud, mixcloud, indieshuffle and youtube... the content in there is awesome, but there is no good way to track awesomeness. My goal is to create a list of regular podcasts,  and mix them according to current mood, or to receive a notification when something new is just posted on a platform.

The collection can grow drastically for sure, but we can easily control and limit it, by sorting it out. For example, a soundcloud weekly podcast can receive bad/good impressions, and thus move around.

If anyone is interested to talk about the possibilities of collecting intelligence from music channels, you can join the chat here: https://collectingmusic.slack.com/messages
